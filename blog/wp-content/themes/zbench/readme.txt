== Description ==
It is a simple WordPress theme without any plugin needed (support plugin WP-PageNavi, WP Page Numbers, WP-PostViews, wp-utf8-excerpt, Related Posts of Simple Tags, WP-RecentComments), little images, custom-menu, widgets, threaded-comments. For WordPress version 3.0+

== Installation ==

= Via WordPress Admin =
1. From your sites admin, go to Themes > Install Themes
2. In the search box, type 'zBench' and press enter
3. Locate the entry for 'zBench' (there should be only one) and click the 'Install' link
4. When installation is finished, click the 'Activate' link

= Manual Install =
1. Unzip this file into your mywebsite.com/wp-content/themes/ directory, should look like mywebsite.com/wp-content/themes/zbench/
2. Go into your Wordpress Admin, navigate to 'Appearance > Themes'
3. Find the zBench listing on this page and click 'Activate'
