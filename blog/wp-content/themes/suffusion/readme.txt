== Description ==

An elegant, versatile and browser-safe theme with a power-packed set of options. It has 19 widget areas, one-column, two-column and three-column
fixed-width and flexible-width formats, 10 pre-defined templates, 19 pre-defined color schemes, two customizable multi-level drop-down menus,
featured posts, a magazine layout, tabbed sidebars, widgets for Twitter, Social Networks and Google Translator, translations in many languages
and RTL language support. Native WP Menus are integrated and additional support is provided for Mega Menus. A BuddyPress support pack is available as a
plugin for smooth BuddyPress integration. Support forum at http://www.aquoid.com/forum.

= Note =
*   The theme supports custom headers through its admin panels: Suffusion Options -> Skinning -> Header
*   The theme has special support for the Chat, Status, Audio and Gallery post formats, in addition to the Standard post format.

== Installation ==

You can install the theme through the WordPress installer under <strong>Themes &rarr; Install themes</strong> by searching for it.
Alternatively you can download the file from here, unzip it and move the unzipped contents to the <code>wp-content/themes</code> folder
of your WordPress installation. You will then be able to activate the theme.

== Frequently Asked Questions ==

= Where can I get support for the theme? =
Support is available at the <a href='http://www.aquoid.com/forum'>theme's support forum</a>.

= Does Suffusion work with BuddyPress? =
Yes. There is a plugin called <a href='http://wordpress.org/extend/plugins/suffusion-buddypress-pack/'>Suffusion BuddyPress Pack</a>,
more information about which can be found <a href='http://www.aquoid.com/news/plugins/suffusion-buddypress-pack/'>here</a>.

= Does Suffusion work with e-commerce plugins? =
There is a plugin called <a href='http://wordpress.org/extend/plugins/suffusion-commerce-pack/'>Suffusion Commerce Pack</a>,
which extends Suffusion support to Jigoshop. Addition plugins are planned for future releases.

= Does Suffusion work with bbPress? =
Not at this point. Support will be added in a later release.

= Are translations supported? =
Translations for the theme are available at http://aquoid.com/news/themes/suffusion/translating-suffusion/. The back-end of the theme is
not currently translated. However a plugin will be made available to facilitate translations of the same.

= What languages is Suffusion available in? =
There are several translations distributed <a href='http://www.aquoid.com/news/themes/suffusion/translating-suffusion/'>here</a>.

= Where can I get more information about the sidebars/widget areas in Suffusion? =
Suffusion has multiple widget areas at different locations. For a diagram of the sidebar layout go to Suffusion Options &rarr; Sidebars &rarr; Sidebar Layout.

= How do Mega Menus work? =
To use a mega-menu, create a regular WP menu, then add items to this menu. Every top-level menu item will provide you with the option
to assign a widget area for use as a mega-menu. Pick a widget area of your choice and add widgets to it from <em>Appearance &rarr; Widgets</em>.
Note that the mega-menu width is fixed, computed based on the number of columns you pick. The width will not auto-adjust based
on the actual number of widgets in the widget area.
