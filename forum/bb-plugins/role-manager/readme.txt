== Description ==
Role Manager allows key masters to change the capabilities that each role has and even create new roles.

== Installation ==
1. Upload the entire `role-manager` plugin to the `my-plugins` directory of your bbPress installation. If you don't have one, create it!
2. Activate the plugin in your administration panel.
